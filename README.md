# Καθημερινά V14.0.5 — Stable Learning Path UX Polish

A small, stable upgrade from V14.0.4F. This build stays on the V14 line and keeps `const LS='gta_v12_state'`.

## Daily path

Today → Study the exact concept in Library → Practice the examples → Start the matching worksheet → Grade → Reveal Greek/English/choice meanings/glossary → Add unknown words to Review when needed.

## V14.0.5 changes

- Today now shows one next lesson and one main action.
- Today, Library, and Worksheets share one lightweight current-concept pointer.
- Levels is clearly labeled as a roadmap; topic chips are visibly non-clickable labels.
- “See concept sequence” expands to real **Study lesson** and **Start worksheet** actions.
- Worksheet results now show correct/wrong status, Greek transcript or sentence, English meaning, answer-choice translations, a glossary, and unknown-word Review actions.
- Review is simplified to Unknown Words, Weak Spots, Mistakes, Recordings, and Completed Items.
- Review remains recommended and never blocks new study.
- Buttons and tabs have visible tap feedback.
- Service worker cache updated to `gta-v14-0-5-stable-learning-path` with network-first document loading and safe old-cache cleanup.

## Stability boundary

No V15 rebuild, no heavy Kumon locks, no complicated router, no Library/Worksheet redirect loop, and no state-key change.
